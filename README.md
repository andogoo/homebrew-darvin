# Homebrew Tap — Darvin Core

Official Homebrew tap for [Darvin Core](https://github.com/andogoo/darvin-core).

## Install

```bash
brew tap andogoo/darvin
brew install darvin
darvin setup
```

## Update

```bash
brew update && brew upgrade darvin
```

## Requirements

- macOS (Apple Silicon or Intel)
- Homebrew
- Xcode Command Line Tools

## What you get

- `darvin` CLI (setup wizard, doctor, lifecycle commands)
- Node 20 runtime (auto-installed)
- tmux, git (auto-installed)
